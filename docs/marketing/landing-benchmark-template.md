# Benchmark de landing (template anônimo)

Preencha **fora do Git** com URLs reais se quiser rastrear marcas; neste arquivo versionado use apenas `URL_1`, `URL_2`, … e **categorias**. Não inclua nomes de produtos ou empresas concorrentes aqui.

## 1. Categorias que você está comparando

Exemplos de categoria (ajuste os rótulos):

| ID | Categoria |
|----|-----------|
| C1 | Widget ou extensão de relato de bug + captura de tela |
| C2 | Gravação de sessão / replay para suporte ou produto |
| C3 | Feedback visual + heatmaps ou analytics |
| C4 | Enterprise / foco em compliance ou escala |
